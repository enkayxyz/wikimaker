# WikiMaker

WikiMaker is a corpus-to-wiki scaffold for recursive Markdown trees.

This first version is intentionally conservative:
- scan corpus Markdown files
- compute change detection
- write telemetry
- generate AI-backed source-summary pages
- identify commonality and synthesize wiki sets
- maintain folder gist.md and ledger.md memory
- keep ADK in the orchestration/observability layer

For the detailed requirements, see `docs/wikimaker/requirements.md`.
For usage, see `docs/wikimaker/runbook.md`.
