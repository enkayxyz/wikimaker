# Corpus Profile Skill

Use corpus family metadata to choose concise, relevant SourceCard fields. Keep WhatsApp chats, AI conversations, financial documents, contacts, calendars, meeting notes, transcripts, emails, code, project artifacts, index pages, and ledger pages distinct unless the cards provide clear relationships.

Prefer deterministic metadata, headings, links, dates, and amounts before asking the model to infer semantic labels.
